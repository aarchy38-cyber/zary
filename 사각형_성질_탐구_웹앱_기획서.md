# 사각형의 성질 탐구·변형 웹앱 기획서

## 1. 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 프로젝트명 | (가칭) 사각형 실험실 (Quadrilateral Lab) |
| 대상 | 중학교 수학 — 평면도형 단원 (사각형의 성질) |
| 핵심 컨셉 | 학생이 꼭짓점을 직접 드래그하며 사각형을 변형하고, **변의 길이·각도·대각선 관계가 실시간으로 어떻게 바뀌는지** 눈으로 확인하면서 사각형 사이의 포함 관계(정의-성질-판정 조건)를 스스로 발견하는 동적 기하 도구 |
| 차별점 | 단순 작도 툴이 아니라, "조건을 잠그고(Lock) 변형 → 자동 판별 → 가족 관계 시각화 → 도전 과제"로 이어지는 **탐구형 학습 흐름**을 제공 |

---

## 2. 교육 목표 (학습 목표)

중학교 1학년 평면도형 성취기준(예: 9수03-08~10 사각형의 성질 관련)을 토대로 설정합니다.

1. 평행사변형이 되기 위한 조건(대변 평행/길이/대각/대각선)을 직접 변형을 통해 발견한다.
2. 직사각형, 마름모, 정사각형이 평행사변형의 어떤 조건을 추가로 만족하는 특수한 경우인지 이해한다.
3. 사다리꼴과 등변사다리꼴의 성질(평행, 밑각, 대각선 길이)을 비교한다.
4. 사각형 사이의 포함 관계(정사각형 ⊂ 직사각형 ⊂ 평행사변형, 정사각형 ⊂ 마름모 ⊂ 평행사변형)를 벤다이어그램·트리로 시각화하여 이해한다.
5. "이 사각형을 마름모로 만들려면 무엇을 바꿔야 할까?" 같은 역방향 추론(변형 미션)을 통해 성질을 능동적으로 적용한다.

## 3. 핵심 기능

### 3.1 자유 변형 캔버스 (Free Transform Mode)
- 임의의 사각형 4개의 꼭짓점(A, B, C, D)을 화면에 표시, 각 점을 마우스/터치로 드래그 가능
- 드래그 중 실시간으로 다음을 자동 계산·표시:
  - 네 변의 길이 (AB, BC, CD, DA)
  - 네 내각의 크기 (∠A, ∠B, ∠C, ∠D, 합 360° 표시)
  - 대변의 평행 여부 (기울기 비교, 평행 시 화살표 기호 ▶▶ 자동 표시)
  - 대각선 AC, BD의 길이, 교점, 수직 여부, 이등분 여부
- 변형과 동시에 하단/측면 패널에 "현재 이 도형은 **사다리꼴**입니다" 처럼 **실시간 자동 분류 결과**와 판별 근거(만족 조건 목록)를 출력

### 3.2 조건 고정 변형 (Constraint Lock Mode)
동적 기하의 핵심 기능 — 특정 성질을 "잠금(Lock)"하면, 그 이후 점을 드래그해도 잠긴 조건은 항상 유지되도록 자동 보정합니다.

| 잠금 가능한 조건 | 잠금 시 동작 |
|---|---|
| 대변 평행 (한 쌍 / 두 쌍) | 한 변을 움직이면 평행한 변이 함께 회전/이동 |
| 대변 길이 같음 | 변의 길이를 동일하게 자동 유지 |
| 네 변 길이 같음 | 마름모 형태를 유지하며 변형 |
| 네 각 직각 | 직사각형 형태를 유지하며 크기만 변형 |
| 대각선 수직 | 대각선이 항상 90°를 이루도록 유지 |
| 대각선 길이 같음 | 등변사다리꼴/직사각형 탐구에 활용 |

- 여러 조건을 동시에 잠그면 자유도가 줄어드는 것을 체험 → "조건을 추가할수록 더 특수한 사각형이 된다"는 개념을 체득
- 조건 잠금 시 잠금 아이콘과 함께 "지금 어떤 변의 정의 조건을 만족 중인지" 안내 문구 표시

### 3.3 사각형 가족 관계 트리 (Hierarchy Explorer)
- 사각형 → 사다리꼴 → 평행사변형 → {직사각형, 마름모} → 정사각형 으로 이어지는 **포함 관계 트리**를 인터랙티브 다이어그램으로 제공
- 트리의 각 노드를 클릭하면 해당 도형의 정의·성질·판정 조건이 카드로 펼쳐짐
- 노드 간 화살표를 따라가며 "정사각형은 왜 직사각형이기도 하고 마름모이기도 한가?"를 시각적으로 설명
- 자유 변형 캔버스에서 만든 도형이 트리 위에 실시간으로 하이라이트되어 "지금 내가 만든 도형의 위치"를 보여줌

### 3.4 변형 미션 모드 (Challenge Mode)
- 단계별 미션 제공, 예시:
  1. "평행사변형을 만들어보세요." → 조건 충족 시 ✅ 통과 애니메이션
  2. "방금 만든 평행사변형을 변형 없이 직사각형으로 만들려면 어떤 조건을 추가해야 할까요?" (객관식 또는 직접 변형으로 증명)
  3. "대각선이 서로 수직이면서 이등분하는 사각형을 만들어보세요." (마름모 발견 유도)
  4. "두 대각선의 길이가 같은 사다리꼴을 만들어보세요." (등변사다리꼴)
- 미션 클리어 시 별점/배지 등 게이미피케이션 요소
- 오답(조건 불충족) 시 어떤 조건이 부족한지 힌트로 제시 (예: "두 변이 아직 평행하지 않아요. 기울기를 비교해보세요.")

### 3.5 측정 및 증명 도구
- 각도기, 자 형태의 측정 오버레이 토글
- 보조선(대각선) 표시/숨김
- 두 변/두 각이 같음을 색칠 표시(합동 표시 ⫽, ⌒ 기호) 기능 — 교과서 증명 표기와 동일하게 학습 가능
- "이 도형이 평행사변형인 이유"를 단계별 논리(증명) 카드로 자동 생성 (예: 대변 평행 2쌍 → 평행사변형의 정의 충족)

### 3.6 교사용 기능
- 학급 코드 생성 → 학생들이 코드로 입장, 교사 화면에 학생별 진행 상황 실시간 모니터링
- 미션 세트 직접 편집(순서/난이도 커스터마이징), 차시별 수업 자료로 저장
- 학생이 만든 도형 캡처 모아보기(과제 제출/형성평가 활용)
- 전체화면 + 빔프로젝터 모드(칠판 판서 대용 시연용, 큰 글씨/고대비)

## 4. 사용자 흐름 (User Flow)

```
[시작 화면] 학생 / 교사 모드 선택
        ↓
[학생] 학급 코드 입력 → 학습 모듈 목록(차시별)
        ↓
[자유 탐구] 또는 [미션 모드] 선택
        ↓
   자유 탐구: 꼭짓점 드래그 → 실시간 성질 패널 확인 → (선택) 조건 잠금
        ↓
   미션 모드: 단계별 과제 수행 → 자동 판별 → 힌트/정답 피드백 → 다음 미션
        ↓
[가족 관계 트리]에서 방금 만든 도형 위치 확인
        ↓
[결과 저장] 캡처/제출 → 교사 대시보드 반영
```

## 5. 화면 구성 (와이어프레임 설명)

1. **메인 캔버스 (중앙, 화면의 60~70%)**: SVG/Canvas 기반 사각형, 꼭짓점은 큰 원형 핸들로 표시(터치 친화적)
2. **속성 패널 (우측 또는 하단)**: 변 길이, 각도, 대각선 정보를 실시간 숫자 + 그래픽 아이콘으로 표시
3. **분류 배지 (캔버스 상단)**: 현재 도형 종류를 큰 글씨 배지로 표시 (예: "🟦 평행사변형")
4. **조건 잠금 토글 패널 (좌측)**: 체크박스 형태로 조건 on/off
5. **가족 관계 트리 (별도 탭/모달)**: 클릭 가능한 다이어그램
6. **미션 사이드바**: 현재 미션 설명, 진행도 바, 힌트 버튼

## 6. 수학적 로직 명세 (판별 알고리즘)

좌표 A(x1,y1), B(x2,y2), C(x3,y3), D(x4,y4) 기준 (사각형 ABCD, 시계/반시계 한 방향으로 입력 가정).

```
변 벡터:
  AB = B - A,  BC = C - B,  CD = D - C,  DA = A - D

변의 길이:
  len(AB), len(BC), len(CD), len(DA)  // 유클리드 거리

평행 판정 (AB ∥ CD):
  cross(AB, CD) ≈ 0   // 외적이 0에 가까우면 평행 (부동소수 오차 허용 ε 적용)

평행사변형 판정 (다음 중 하나라도 만족하면 동치로 처리):
  - AB ∥ CD  AND  BC ∥ DA
  - len(AB) = len(CD)  AND  len(BC) = len(DA)
  - 대각선 AC, BD가 서로를 이등분 (교점이 두 대각선의 중점과 일치)

직사각형 판정:
  평행사변형 조건 + 한 내각이 90° (cos θ ≈ 0)
  ⇔ 평행사변형 + 두 대각선의 길이가 같음 (len(AC) = len(BD))

마름모 판정:
  평행사변형 조건 + 네 변의 길이가 모두 같음
  ⇔ 평행사변형 + 두 대각선이 서로 수직 (dot(AC, BD) ≈ 0)

정사각형 판정:
  직사각형 조건 AND 마름모 조건 (둘 다 만족)

사다리꼴 판정:
  AB ∥ CD  XOR  BC ∥ DA   // 한 쌍만 평행

등변사다리꼴 판정:
  사다리꼴 조건 + 평행하지 않는 한 쌍의 변의 길이가 같음
  ⇔ 사다리꼴 + 두 대각선의 길이가 같음
```

- 모든 비교는 픽셀/좌표 오차를 고려해 허용 오차(ε, 예: 0.5°, 0.5px)를 두고 판정 → "거의 맞았을 때" 스냅(자동 정렬) 옵션 제공 가능
- 자동 분류 패널은 **가장 특수한 분류**(예: 정사각형이면 정사각형이라고만 표시, "평행사변형이기도 함"은 보조 텍스트로 표시)

## 8. 기술 스택 제안

| 영역 | 제안 기술 | 비고 |
|---|---|---|
| 도형 렌더링 | SVG (또는 Canvas) + 자체 좌표 연산 | 드래그 핸들 다루기엔 SVG가 DOM 이벤트 처리에 유리 |
| 기하 연산 | 직접 구현 (벡터 내적/외적) 또는 경량 라이브러리 | 의존성 최소화를 위해 직접 구현 권장 |
| 동적 기하 참고 모델 | GeoGebra의 "제약 기반 드래그" 동작 방식 참고 | 자체 구현 시 점 이동 → 잠긴 조건 재계산 → 좌표 보정 루프 설계 |
| 프론트엔드 프레임워크 | React + TypeScript | 상태(꼭짓점 좌표, 잠금 조건)를 명확히 타입으로 관리 |
| 애니메이션 | Framer Motion (미션 클리어 연출, 트리 펼침 효과) | |
| 상태 관리 | Zustand | 캔버스 상태와 미션 진행 상태 분리 관리 |
| 백엔드 (교사 대시보드용) | Firebase/Supabase | 학급 코드, 학생 진행도 실시간 동기화 |
| 배포 | Vercel/Netlify | 정적 배포로 충분, 학교 네트워크 환경 고려해 가벼운 번들 유지 |

## 9. 비기능 요구사항
- 터치스크린(전자칠판, 태블릿) 환경에서 꼭짓점 드래그가 정확히 동작해야 함 (히트 영역을 시각적 점보다 크게 설정)
- 빔프로젝터 송출을 고려한 고대비 색상 팔레트 및 큰 폰트 모드
- 좌표 계산은 클라이언트에서 즉시 처리되어 네트워크 지연 없이 실시간 반응
- 색약 학생을 고려해 평행/합동 표시를 색상뿐 아니라 기호(화살표, 빗금 개수)로도 구분

## 10. 향후 확장 아이디어
- 삼각형 단원과 연계: 사각형의 대각선으로 나뉜 삼각형의 합동 증명 모듈
- 넓이 공식 탐구 모드: 변형하면서 넓이가 실시간으로 어떻게 바뀌는지 시각화 (평행사변형 넓이 = 밑변×높이 직관적 이해)
- 좌표평면 버전: 좌표를 이용한 사각형 성질 증명(중점 공식, 기울기 공식 활용) 심화 모듈
- 협동 모드: 같은 캔버스에 여러 학생이 동시 접속해 함께 도형을 변형하는 실시간 협업 기능
- AR 모드: 카메라로 실제 사각형 물체를 비추면 변/각을 인식해 분류해주는 확장 기능

## 11. 우선순위 제안 (MVP)
1. 자유 변형 캔버스 + 실시간 변/각/대각선 정보 표시
2. 자동 분류 로직 (평행사변형/직사각형/마름모/정사각형/사다리꼴/등변사다리꼴)
3. 가족 관계 트리 (정적 다이어그램 + 클릭 설명)
4. 미션 모드 3~5개 기본 세트
5. 이후 단계에서 조건 잠금(Constraint Lock), 교사 대시보드, 게이미피케이션 요소 확장
